# Freelancer Earnings & Job Trends Analysis

This project explores freelancer earnings and job trends using Python, SQL, and Power BI. The goal was to analyze freelancer performance, test common assumptions, and identify factors that influence earnings.

## Tools Used

* Python (Pandas, Matplotlib, Seaborn)
* SciPy
* MySQL
* Power BI

## Project Overview

* Cleaned and analyzed 1,950+ freelancer records
* Performed exploratory data analysis (EDA)
* Wrote SQL queries to answer business questions
* Conducted T-tests and Chi-square tests
* Built an interactive Power BI dashboard

## Key Findings

* App Development had the highest average earnings.
* Intermediate freelancers earned more than Experts on average.
* Hypothesis testing found no statistically significant relationship between the selected variables.

## Skills Demonstrated

Data Cleaning, EDA, SQL, Hypothesis Testing, Data Visualization, and Dashboard Development.
